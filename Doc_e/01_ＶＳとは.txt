Visual Studio（以後VSと呼ぶ）は、MIcrosoft社が提供するWindows用アプリケーション開発環境のことです。Visual Studioは2,002年にVS 6.0から、VS .net 2002と名称変更しました。同社の提供する仮想マシン環境.net framework上で動作するプログラムであることを名称でも宣言したのです。以後、2,012までの全てのバージョンが、仮想マシン.net framework上で動作させることを前提として作られています。
　VSには、4種類の言語があります。
　・Visuai Basic .net
　・Visual C++/CLI
　・Visual C#
　・Visual F#
　これらの言語は、言語の別に関わらず、全てMSILと呼ばれる中間言語に翻訳され、.net framework上でのみ動作します。この中間言語を仮想マシン上で動かす他言語として、Javaが有名ですが、実際に.net frameworkはJava仮想環境に非常に良く似ており、

・Java 言語　→　VB.net VC++ VC#
・Java VM　→　.net Framework
・Eclipse　→　Visuai Studio

であると読み替えることで、よりよく理解できるかもしれません。
　VSの画面は、かつてのVS 6.0を踏襲しており、Windows用のフォームアプリケーションをビジュアルに、かつスピーディに開発することができます。現在では、インテリジェンス昨日などのプログラム自動補完機能も充実しており、初心者からすぐにコーディングが始められます。
　他方で、Windowsフォームの基盤を.net frameworkに変更したことで、MFCなど従来のVS 6.0のネイティブアプリケーション（独自アプリ）をそのまま流用することができなくなりました。　かつてのVBと、VB.netは文法上は似ていても、まったく別のプログラム言語だと捕らえておくほうが無難でしょう。
　ただし、VSの4言語で書かれたアプリケーションは、.net frameworkの搭載されているWindowsXP以降、全ての端末で動作させることができます。つまりＶＢで作成したライブラリをＣ＃で参照して使用するなどの、言語の垣根を越えたアプリケーション開発が可能なのです。
